# 🔗 UrlShortCut

**Сервис сокращения ссылок** с JWT аутентификацией и статистикой переходов.

---

## 📋 О проекте

UrlShortCut — это микросервис, который позволяет сайтам:
- 🔐 Регистрироваться и получать API-ключи
- ✂️ Превращать длинные ссылки в короткие коды
- 📊 Отслеживать статистику переходов
- 🔄 Перенаправлять пользователей по коротким ссылкам (HTTP 302)

**Предназначен для интеграции с другими сайтами через REST API.**

## 📦 Установка и запуск

### Требования
- Java 11 или выше
- Maven 3.6+

### Шаги

```bash
# Клонируем репозиторий
git clone https://github.com/your-username/job4j_url_shortcut.git
cd job4j_url_shortcut

# Собираем проект
mvn clean package

# Запускаем
mvn spring-boot:run
